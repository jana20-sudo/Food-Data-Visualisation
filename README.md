🍽️ Food Data Visualization Dashboard

📌 Overview
This project presents an interactive dashboard for visualizing food-related data using web technologies.

🛠️ Technologies Used
- HTML
- CSS
- JavaScript

📊 Features
- Interactive charts
- Dashboard layout
- User-friendly visualization

📁 Files
- index.html → Main page
- dashboard.html → Dashboard view
- style.css → Styling
- main.js → Logic and interactions

🚀 How to Run
1. Download the repository
2. Open index.html in your browser

📌 Purpose
To explore and present food data insights visually in a web-based dashboard.

📈 Future Improvements
- Add more interactivity
- Connect with real datasets
